Experiment Note

At 0% noise, this system is fully deterministic: the same seed and the same B3/S23 rule produce the same outcome every run, analogous to an LLM at temperature 0. Raising execution noise is a loose analog of raising sampling temperature: it introduces run-to-run divergence without changing the starting seed or the rule.
